프로세스 = 메모리에서 해당 프로그램(스태틱/스택/힙 등의 영역)을 실행하는 것
--> 앱 하나 실행할 때마다, 프로세스가 실행됨
(애플리케이션 = 하나의 프로그램 단위)

* 멀티 프로세스
= 하나의 앱에, 여러 개의 프로세스가 실행
= 프로세스 간 서로 영향을 주지 않음

스레드(경량 프로세스) = 프로세스 내부에서 실행되는, 하나의 코드 실행 단위

* 멀티 스레드 = 스레드 간 서로 영향을 줌(따라서, 예외 처리가 필요함)
--> 스택은 스레드 하나당, 하나씩 생성
--> 코드, 스태틱, 힙 등 공유

싱글 스레드 = 코드가 순차적으로 실행
멀티 스레드 = (여러 track에서)코드가 동시다발적으로 실행

스레드 안전 = 여러 스레드로부터 동시에 접근을 하더라도, 순차적으로 실행하게 만드는 것(하나의 스레드만 접근할 수 있도록 처리)
--> synchronized 메서드()으로 구현 가능
