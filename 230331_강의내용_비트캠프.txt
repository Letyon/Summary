<while문 logic>
조건식을 검사해서 조건식이 참이면 실행문을 반복하고, 거짓이면 while문을 빠져나옵니다.
실행할지 말지는 시작 지점(조건문)에서 판단함.
while문 끝나는 부분을 만나면, 다시 시작 지점(조건문)으로 올라가야 함.

do while문 = 최소 1번 이상 실행(그 이후로는 while문과 동일)
while문 = 0번 실행되는 경우 있음

for(초기값; boolean형 조건식; 증감식) {
    반복할 실행문;
}

for(int i = 0; i < 5; i++)
-> 공식 : i = 0인 상태에서 i < n (i는 n바퀴 회전함)
-> i의 경우의 수는 5개이므로, i는 5바퀴 회전함

파이썬튜터 = frames(스택) objects(힙)

공식 : i % n == 0이면, (i는 n의 배수임)

향상된 for 루프(배열 처리에 많이 사용)
for(배열의 값을 담을 변수 : 배열의 이름) {
     실행문;
}
